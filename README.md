# Taller de Aptitudes Lógicas y Matemáticas (TALM)

Este repositorio contiene todo el material desarrollado y recopilado durante el ciclo de la asignatura **Taller de Aptitudes Lógicas y Matemáticas (TALM)**, incluyendo apuntes de clases, evaluaciones resueltas y el código fuente del proyecto final.

## Estructura del Repositorio

El material ha sido curado y organizado en las siguientes secciones lógicas:

*   **`[Apuntes/](file:///Apuntes)`**: Contiene las notas de estudio y apuntes teóricos de la asignatura.
*   **`[Controles/](file:///Controles)`**: Evaluaciones parciales. Incluye el **Control 2** (Cinemática de un Brazo Robótico 3D) con su código fuente en LaTeX, imágenes asociadas y el PDF definitivo.
*   **`[Proyecto_Hito/](file:///Proyecto_Hito)`**: Proyecto semestral (Desafío Matemático 2026).
    *   **`[Documentos/](file:///Proyecto_Hito/Documentos)`**: Rúbricas, pautas oficiales, casos de estudio (Ingeniería y Salud) y formatos base proporcionados por la USS.
    *   **`[Hito_Entregables/](file:///Proyecto_Hito/Hito_Entregables)`**: Informes escritos y presentaciones finales del proyecto (en formato PDF y código fuente LaTeX). Contiene tanto la propuesta inicial como las entregas de hitos finales.
    *   **`[Scripts/](file:///Proyecto_Hito/Scripts)`**: Scripts de Python utilizados para la simulación 3D del brazo robótico y la generación de gráficos de stress/análisis.
*   **`[Gemini_Context/](file:///Gemini_Context)`**: Bitácora y prompts de IA utilizados para guiar el desarrollo de los algoritmos de simulación y LaTeX.

---

## Requisitos y Ejecución de Scripts

Para ejecutar las simulaciones y generación de gráficos, requiere tener instalado Python 3 con las librerías estándar y adicionales de visualización.

Ubicación de los scripts:
*   [simulador_3d.py](file:///Proyecto_Hito/Scripts/simulador_3d.py)
*   [generate_plots.py](file:///Proyecto_Hito/Scripts/generate_plots.py)
